# Homebrew Tap

Custom Homebrew tap for Record Screen.

## Install

```bash
brew tap faker6996/tap
brew install --cask faker6996/tap/record-screen
```

## Repository layout

```text
homebrew-tap/
├── Casks/
│   └── record-screen.rb
└── .github/
    └── workflows/
        └── validate-cask.yml
```

This tap is updated automatically by the main `record-screen` release workflow.
